# 矽谷亞洲青年創新領袖峰會 2026 — 活動回顧與事實查核

> 2026-07-19 台大蘇格拉底廳「矽谷亞洲青年創新領袖峰會」的個人活動整理：經查證的活動資訊、講者背景逐項事實查核（附來源）、與當天聽講心得。

我參加了這場峰會，把主辦方文案裡的每一項聲稱逐項與獨立來源（官方網站、政府與公司登記、SEC 文件、媒體報導）比對後做成這個網站。每項聲稱標注三種狀態之一：✅ 已驗證、⚠️ 部分吻合、❌ 僅主辦方宣稱；完整查核筆記在 [`research/`](research/)。

## ✨ 功能特色

- 🌏 **雙語切換** — English / 中文一鍵切換，整頁重繪無殘留
- 🌗 **深色 / 淺色模式** — 編輯風雙主題，選擇會被記住
- 🏷️ **查證徽章** — 每位講者、每項聲稱標注「已驗證 / 部分吻合 / 僅主辦方宣稱」
- 🔗 **深連結** — 每位講者都有專屬 `#<slug>`，可直接分享查核詳情
- 📝 **聽講心得區** — 每場分享一格，陸續補上
- 📱 **響應式設計** — 手機、平板、桌機皆適配
- ⚡ **純靜態** — 無後端、零 build、載入快

## 📂 內容結構 / 資料來源

```
sv-asia-summit-2026/
├── index.html        # 入口頁
├── assets/           # 樣式與前端互動邏輯
├── data/data.js      # 資料層（活動、講者、查核狀態、心得）
└── research/         # 完整查核筆記（逐項比對表 + 全部來源連結）
```

本站內容整理自 **Accupass 活動頁、主辦方公開貼文，以及各講者的官方網站、公司登記、SEC 文件與媒體報導**（完整來源清單見 `research/` 各檔與網站內的參考連結）。

> ⚠️ **非官方**：本網站為個人整理之非官方資源，與主辦單位無關。查證狀態反映 2026-07-20 當時可取得的公開資訊；如有錯誤或出入，歡迎開 issue 指正。

### 補充聽講心得

`data/data.js` 的 `window.SITE_NOTES` 中，把對應講者的 `content: null` 改成雙語內容即可發布，例如：

```js
{ speaker: "andy-lin", content: { en: ["First paragraph…"], zh: ["第一段…"] } }
```
